# jQuery tooltip plugin

Usage:

Include source and styles files
```html
<link rel="stylesheet" type="text/css" href="tooltip.css" media="all">
<script src="tooltip.js"></script>
```

Initialize
```html
$('.some-target').tooltip('.content-container');
```
